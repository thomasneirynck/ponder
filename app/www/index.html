How does it work?

Ponder uses a neural network called a "self-organizing map" to layout the data in two dimensions. During the training of the network, items that are similar move closer to each other, while items that are dissimilar move away from eachother.

Ponder is an example of unsupervised learning. This means it can detect clusters in the data, without a human having to define which type of clusters is should look at beforehand. You need to provide is what fields contain numerical values, and what fields contain categorical data.

Using the map tools

The tools below are available on the map. Below is a short explanation of how each of them works.
Toggle between the map and the table view. The table shows the features that are selected on the map.
Exaggerate or reduce the contrast on the map. This helps you to identify clusters of data.
For numeric values, use the classification slider to color the features by what side of the break they fall (smaller or larger).
Boxplots shows the distribution for a particular numerical variable of the currently selected features.
Histograms shows the distribution for a particular categorical variable of the currently selected features.
